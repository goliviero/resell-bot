Telegram

Via .env

Telegram est configure via les variables d'environnement TELEGRAM_BOT_TOKEN et TELEGRAM_CHAT_ID dans le fichier .env. Utilise python scripts/setup_telegram.py pour configurer.

Comportement des notifications:
  • Alerte instantanee — envoyee des qu'un deal est detecte (prix ≤ budget). Cooldown 24h par livre pour eviter le spam.
  • Recap journalier — 1 message a 08:00 avec tous les deals actuellement disponibles. 1 post Discord + 1 email + 1 message Telegram.
